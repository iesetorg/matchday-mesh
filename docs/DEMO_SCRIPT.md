# Demo Script

Target length: 3 minutes.

1. Open Matchday Mesh from PearBrowser.
2. Show the app status strip: Pears ready, WDK demo, QVAC gated.
3. Create "Final Night Fan Zone".
4. Claim a fan pass.
5. Show the generated pass QR block.
6. Scan/check in the pass.
7. Post a prediction.
8. Post a match note into the feed.
9. Export the P2P invite and show the Corestore key.
10. Open the USDt pool demo.
11. Add a contribution and show the pool contribution feed card.
12. Open the proof pack.
13. Close with the release link, repo, and track claims.
