# Prior Work Disclosure

Matchday Mesh is a new football-specific app, but it intentionally reuses
patterns from the existing Pear ecosystem.

## Reused Or Adapted Patterns

- Pear Tickets: event/pass/check-in concepts, Pear app packaging shape, catalog
  manifest shape, and publish validation approach.
- Pear POS: WDK-shaped payment adapter direction for USDt receive/pool flows.
- anonGPT native experiments: QVAC local inference direction, only if the QVAC
  SDK path is working before track lock.
- PearBrowser and HiveRelay: app catalog, publish, release, and seed flow.

## Not Yet Claimed

- Production real-money settlement.
- Cloud-free QVAC inference.
- Live sports data.
- Fresh multi-relay availability proof.

Those claims require separate proof before appearing in the DoraHacks
submission.
